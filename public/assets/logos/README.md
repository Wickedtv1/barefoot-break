# Logo Assets

This directory contains the Barefoot Break brand logos.

## Required Logo Files

### Primary Logo
- **`logo-circle.png`** - Circular badge-style logo
  - Used in: Navigation header, Footer
  - Recommended size: 500x500px
  - Format: PNG with transparent background
  - Should include the circular sunset design mentioned in brand guidelines

### Secondary Logo
- **`logo-square.png`** - Square logo with sunset scene
  - Used in: Homepage hero section, Marketing materials
  - Recommended size: 800x800px
  - Format: PNG with transparent background

## Brand Colors (From Logo)

The logos should incorporate these brand colors:
- Deep Burnt Orange / Amber: `#C4550A`
- Golden Yellow: `#D4A017`
- Dark Brown / Shadow: `#1A0A00`
- Warm Sand / Cream: `#FFF3D6`
- Ocean Teal (accent): `#1A8A8A`

## Usage Notes

- Always use PNG format with transparency for web
- Never stretch or distort the logos
- Maintain adequate whitespace around the logos
- Use the circle logo for navigation (fits better in limited space)
- Use the square logo for hero sections and featured placements
